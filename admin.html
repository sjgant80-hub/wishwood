① Domain & site TO SET UP

Where the public-facing site lives. Right now it's on GitHub Pages under sjgant80-hub.github.io/wishwood/. To use wishwood.co.uk instead, point the domain at it.

◌
Current URL: sjgant80-hub.github.io/wishwood/
Working · but not branded · move to wishwood.co.uk when ready
1 Buy the domain
  1. Open workspace.google.com · if you're doing the Google Workspace step (email) too, this comes bundled and is the easiest path.
  2. Or buy standalone from Namecheap · ~£10/yr · just the domain, no extras.
  3. Domain to grab: wishwood.co.uk
2 Point it at GitHub Pages
  1. In your domain registrar (Google or Namecheap), open the DNS / nameserver settings for wishwood.co.uk
  2. Add these 4 A records (one at a time): 185.199.108.153 · 185.199.109.153 · 185.199.110.153 · 185.199.111.153 · all pointing to @ (the root)
  3. Add a CNAME record for www pointing to sjgant80-hub.github.io
  4. Wait 5-30 min for DNS to propagate · check with whatsmydns.net
3 Tell GitHub Pages the new domain
Once DNS is live, paste here. Saves a CNAME file to the repo on next push.
◊ tip
Use HTTPS-enforce in repo settings → Pages once domain is verified. GitHub provisions free SSL automatically.

② Email · Google Workspace TO SET UP

Real business email at chrissy@wishwood.co.uk. Includes Gmail, Calendar, Drive. Pulls Airbnb/Booking messages into the hub via IMAP.

1 Sign up for Workspace
  1. Go to workspace.google.com · click Get started
  2. Plan: Business Starter (~£5.20/user/month)
  3. Use the wishwood.co.uk domain (from step ①)
  4. Set up the first user: chrissy@wishwood.co.uk
  5. Recovery: your personal mobile + personal Gmail
2 Connect to the hub

The hub fetches Airbnb / Booking messages from Gmail using IMAP. To enable:

  1. In Gmail · gear icon → Settings → Forwarding and POP/IMAP → enable IMAP
  2. Create an app password: myaccount.google.com/apppasswords (needs 2FA enabled first)
  3. Paste the app password below
16 chars, stored encrypted as a Cloudflare Worker secret · never visible after save
3 Auto-forward Airbnb & Booking emails
  1. In Airbnb host settings → Account → Notifications → ensure all messages forward to chrissy@wishwood.co.uk
  2. In Booking.com extranet → Account → Email → set primary inbox to the same Gmail
  3. The hub IMAPs every 60s · auto-parses sender + message · routes to unified inbox
